# Consumables Data

This is the master registry for all consumable items defined in `GameConstants.js`.

| ID | Type | Effect |
| :--- | :--- | :--- |
| `tiny_hp_potion` | `HEAL_HP` | Restores 30% of Max HP |
| `tiny_mp_potion` | `HEAL_MP` | Restores 30% of Max MP |
| `teleport_scroll` | `ESCAPE` | Immediately ends battle (escapes) |
